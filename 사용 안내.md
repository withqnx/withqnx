# 겸손몰 후기 분석 — 사용 안내

## 어떻게 보나요?

1. 파인더에서 NAS의 **Mall(new) → DATA** 폴더 열기
2. **`겸손몰 후기 분석.html`** 파일을 더블클릭
3. 기본 브라우저에서 자동으로 열립니다.

## 무엇을 볼 수 있나요?

- **Today** — 오늘 새로 들어온 후기
- **Analysis** — 상품별/그룹별 후기 분포, 키워드 분석
- **Groups** — 상품들을 묶어서 출시일 기준 분석 (각자 설정)
- **Review** — AI가 확신 못한 후기 검토
- **Photos** — 후기 첨부 사진 갤러리

## 데이터는 언제 업데이트되나요?

매일 (담당자가 크롤링 후 자동 배포). 파일 수정 시각 확인하면 됩니다.

## 그룹 설정이나 분류 수정은?

각자 브라우저에 저장됩니다. 다른 사용자에게는 공유되지 않습니다.
브라우저를 바꾸거나 시크릿 모드로 열면 초기화됩니다.

## 안 열리거나 빈 화면이면

- 브라우저: Chrome, Safari, Edge 등 최신 버전 권장
- 파일 크기가 커서(10MB 이상) 처음 로딩에 5-10초 걸릴 수 있음
- 그래도 안 되면 담당자에게 문의
